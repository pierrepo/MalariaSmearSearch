Malaria Smear Search is a web application designed to upload and annotate thick and thin blood smears used for Malaria diagnosis.

Malaria Smear Search also proposes a training module for students, technicians and physicians involved in Malaria diagnosis.

This application is open source and all source code is available on GitHub. All data are under Creative Commons Attribution 4.0 International (CC BY 4.0) license.
